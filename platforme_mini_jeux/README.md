# Platforme de mini jeux

Application web développée en HTML, CSS, JavaScript et PHP.
Le projet fonctionne en environnement local avec XAMPP.

## Technologies utilisées :
- HTML / CSS
- JavaScript
- PHP
- MySQL (via phpMyAdmin)
- XAMPP (Apache + MySQL)

## Prérequis :
- XAMPP installé
- Navigateur web

## Installation et Execution :
### Cloner le projet :
```bash
git clone https://github.com/ton-username/nom-du-repo.git
```

### Copier le dossier du projet dans :
C:\xampp\htdocs\

### Lancer XAMPP :
-Démarrer Apache
-Démarrer MySQL
