# Platforme de mini jeux

Application web développée en HTML, CSS, JavaScript et PHP.
Le projet fonctionne en environnement local avec XAMPP.

## Technologies utilisées :
- HTML / CSS
- JavaScript
- PHP
- MySQL (via phpMyAdmin)
- XAMPP (Apache + MySQL)

## Prérequis :
- XAMPP installé
- Navigateur web

## Installation et Execution :

### Cloner le projet :
```bash
git clone https://github.com/HassanCharaf25/platforme_mini_jeux/
```

### Copier le dossier du projet dans :
C:\xampp\htdocs\

### Lancer XAMPP :
- Démarrer Apache
- Démarrer MySQL

### Ouvrir le projet dans le navigateur :
http://localhost/platforme_mini_jeux/code_source/index/
